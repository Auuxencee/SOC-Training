# Core — Infrastructure SOC de base

Ce dossier contient le **squelette fonctionnel** de ton SOC. C'est ta fondation : tu vas l'enrichir au fil des niveaux.

## Contenu

| Dossier | Description |
|---------|-------------|
| `config/` | Fichiers de configuration Wazuh à compléter |
| `docs/` | Documentation pédagogique sur le SOC |
| `incident-response/` | Playbooks et checklist de réponse aux incidents |

## Par où commencer ?

1. Lis `docs/01-soc-overview.md` pour comprendre ce qu'est un SOC
2. Explore `docs/02-tools.md` pour maîtriser les outils en place
3. Consulte `config/` — les fichiers sont volontairement minimalistes, c'est normal
4. Parcours `incident-response/` pour te familiariser avec la méthodologie

> **Principe clé** : le core évolue avec toi. Après chaque niveau, tu reviendras ici pour ajouter des règles, affiner les configs, compléter les playbooks.
